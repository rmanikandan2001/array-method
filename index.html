Array Methods in Table
method programm/code output
Array length

returns the length (size) of an array

 let num = [1,2,3];
console.log(num.length); 		output : 6
Array to string()

converts an array to a string of (comma separated) array values

 let num = [1,2,3];
console.log(num.toString()); 		output : 1,2,3
Array to pop()

removes the last element from an array

 let num = [1,2,3];
console.log(num.pop());
console.log(num); 		output : 3
[1,2]
Array push()

adds a new element to an array (at the end)

 let num = [1,2,3];
console.log(num.push(4));
console.log(num); 		output : 4
[1,2,3,4]
Array shift()

removes the first array element and "shifts" all other elements to a lower index

 let num = [1,2,3,4];
console.log(num.shift());
console.log(num); 		output : 1
[2,3,4]
Array unshift()

adds a new element to an array (at the beginning), and "unshifts" older elements

 let num = [1,2,3,4];
console.log(num.unshift(0));
console.log(num); 		output : 5
[0,1,2,3,4]
Array join()

returns an array as a string

 let num = [1,2,3,4];
console.log(num.join('|')); 		output : 1|2|3|4
Array delete() Array elements can be deleted using the
JavaScript operator delete.
Using delete leaves undefined holes in the array.
Use pop() or shift() instead. 		output : nill
Array concat()

creates a new array by merging (concatenating) existing arrays

 let num = [1,2,3];
let number = [4,5];
console.log(num.concat(number)); 		output : [1,2,3,4,5]
Array flat()

creates a new array with sub-array elements concatenated to a specified depth

 let num = [[1,2],[3,4]];
console.log(num.flat()); 		output : [1,2,3,4]
Array splice()

can be used to add new items to an array

 let num = [1,2,3,4];
console.log(num.splice(1,3)); 		output : [2,3,4]
Array slice()

slices out a piece of an array

 let num = [1,2,3,4];
console.log(num.slice(1,3)); 		output : [2,3]
